# FocusFit

## Observações sobre o projeto

Visando desenvolver minhas habilidades criei este simples site fictício de academia, utilizando a linguagem de marcação HTML5, linguagem de estilo CSS3 e JavaScript. Ao decorrer do meu aprendizado estarei atualizando com novas ferramentas e novos estilos, tudo para deixar o site mais responsivo possível.
